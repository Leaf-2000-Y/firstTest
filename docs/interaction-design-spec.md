# 00后人生问题知识图谱 - 交互设计规范文档

## 1. 设计原则

### 1.1 核心设计理念
- **流畅性**: 所有交互都应有平滑的过渡动画
- **反馈性**: 用户操作必须有即时的视觉反馈
- **层次性**: 通过动画和视觉效果建立清晰的信息层级
- **愉悦性**: 微交互设计要带给用户愉悦感

### 1.2 动画设计原则
- **有目的**: 每个动画都应有明确的功能目的
- **一致性**: 相同类型的交互使用一致的动画风格
- **性能优先**: 使用 GPU 加速属性（transform, opacity）
- **可禁用**: 尊重用户的 `prefers-reduced-motion` 设置

## 2. 动画系统规范

### 2.1 缓动函数库

| 名称 | 函数 | 使用场景 |
|------|------|----------|
| linear | `t => t` | 连续循环动画 |
| easeOutQuad | `t => 1 - (1-t)²` | 元素进入、展开 |
| easeOutCubic | `t => 1 - (1-t)³` | 面板滑入、弹窗 |
| easeOutElastic | 弹性公式 | 节点展开、按钮点击 |
| easeOutBounce | 弹跳公式 | 成功提示、庆祝效果 |
| spring | 物理弹簧 | 悬停效果、拖拽释放 |

### 2.2 动画时长规范

| 类型 | 时长 | 说明 |
|------|------|------|
| 微交互 | 150-200ms | 按钮悬停、状态切换 |
| 标准过渡 | 300-400ms | 节点展开、面板切换 |
| 复杂动画 | 500-800ms | 多元素协调动画 |
| 环境动画 | 1000-3000ms | 背景效果、脉冲动画 |

### 2.3 延迟与错开

- **Stagger 延迟**: 50ms 每个元素
- **最大延迟**: 不超过 300ms（避免感知延迟）
- **级联延迟**: 从中心向外扩散

## 3. 交互场景详细规范

### 3.1 节点交互

#### 悬停效果 (Hover)
```
触发: 鼠标进入节点区域
动画:
  - 节点放大: 1.0 → 1.15 (spring, tension: 0.5, friction: 0.6)
  - 发光增强: 脉冲强度 0.5 → 1.0
  - 标签显示: 透明度 0 → 1, 位移 Y: 10px → 0
时长: 300ms
```

#### 点击效果 (Click)
```
触发: 鼠标点击节点
动画:
  - 缩放反馈: 1.0 → 0.95 → 1.0 (easeOutQuad, 150ms)
  - 涟漪效果: 从点击点扩散, 半径 0 → 50px
  - 选中边框: 渐显 + 发光
时长: 200ms
```

#### 双击展开 (Double Click)
```
触发: 双击有子节点的节点
动画序列:
  1. 粒子爆炸: 8个粒子向外扩散 (600ms)
  2. 涟漪效果: 从节点中心扩散 (600ms)
  3. 子节点进入:
     - 从父节点位置弹出
     - 弹簧动画到目标位置 (tension: 0.4, friction: 0.7)
     - 缩放: 0 → 1 (easeOutElastic, 400ms)
     - 透明度: 0 → 1 (easeOutQuad, 300ms)
     - Stagger: 每个子节点延迟 50ms
  4. 连接线生长: 从父节点绘制到子节点 (400ms)
  5. 展开指示器旋转: 0° → 90° (300ms)
```

#### 收起效果 (Collapse)
```
触发: 双击已展开的节点
动画:
  - 子节点淡出: 透明度 1 → 0 (200ms)
  - 展开指示器旋转: 90° → 0° (300ms)
  - 连接线渐隐: 透明度 1 → 0 (200ms)
```

### 3.2 画布交互

#### 平移 (Pan)
```
触发: 拖拽空白区域
动画:
  - 实时跟随鼠标
  - 惯性滑动: 释放后继续滑动 (friction: 0.9)
  - 边界回弹: 超出边界时弹性回弹
```

#### 缩放 (Zoom)
```
触发: 滚轮滚动或双指捏合
动画:
  - 以鼠标位置为中心缩放
  - 缩放级别指示器: 显示当前缩放比例
  - 平滑过渡: easeOutQuad, 200ms
范围: 0.1x - 5x
```

### 3.3 面板交互

#### 详情面板滑入
```
触发: 点击节点
动画:
  - 面板: translateX(100%) → translateX(0), scale(0.95) → scale(1)
  - 背景遮罩: opacity 0 → 0.5
  - 内容 stagger: 每项延迟 30ms
缓动: easeOutCubic
时长: 400ms
```

#### 搜索下拉展开
```
触发: 输入搜索内容
动画:
  - 下拉框: opacity 0 → 1, translateY(-10px) → translateY(0)
  - 结果项 stagger: 每项延迟 30ms, 从顶部开始
  - 高亮文字: 下划线从左到右绘制
缓动: easeOutQuad
时长: 300ms
```

### 3.4 按钮微交互

#### 磁性按钮效果
```
触发: 鼠标靠近按钮
动画:
  - 按钮向鼠标方向移动 (最大 8px)
  - 距离计算: 基于鼠标与按钮中心距离
  - 使用 CSS transform, 性能优化
```

#### 点击波纹效果
```
触发: 点击按钮
动画:
  - 波纹从点击位置产生
  - 半径: 0 → 最大 (基于按钮尺寸)
  - 透明度: 0.5 → 0
  - 使用伪元素实现
时长: 400ms
缓动: easeOutQuad
```

## 4. 视觉效果规范

### 4.1 发光效果 (Glow)

#### 多层发光结构
```
外层环境光:
  - 半径: 节点半径 × 2
  - 颜色: 主题色 + 20% 透明度
  - 渐变: 中心 → 边缘渐隐

中层脉冲光晕:
  - 半径: 节点半径 × (1.3 + 脉冲强度 × 0.2)
  - 颜色: 主题色 + 40% 透明度
  - 脉冲: 正弦波动画, 周期 2s

内层柔和光晕:
  - 半径: 节点半径 × 1.2
  - 颜色: 白色 + 30% 透明度
  - 混合模式: screen
```

### 4.2 玻璃态效果 (Glassmorphism)

```css
背景: rgba(15, 23, 42, 0.9)
背景模糊: blur(30px)
边框: 1px solid rgba(255, 255, 255, 0.15)
内发光: inset 0 1px 0 rgba(255, 255, 255, 0.1)
阴影: 0 8px 32px rgba(0, 0, 0, 0.3)
```

### 4.3 标签样式

```
背景:
  - 渐变: 从 rgba(15, 23, 42, 0.9) 到 rgba(30, 41, 59, 0.9)
  - 圆角: 6px
  - 边框: 1px solid 主题色 + 25% 透明度

文字:
  - 主标签: #e2e8f0, 12px
  - 描述文字: #94a3b4, 10px
  - 悬停时: #ffffff
```

## 5. 性能优化规范

### 5.1 GPU 加速
- 使用 `transform` 替代 `top/left`
- 使用 `opacity` 替代 `visibility`
- 添加 `will-change` 提示（动画前添加，动画后移除）

### 5.2 渲染优化
- 使用双缓冲（Offscreen Canvas）
- 限制同时运行的动画数量（最多 20 个）
- 使用对象池复用对象

### 5.3 事件优化
- 使用 `passive: true` 监听滚动事件
- 节流/防抖高频事件（resize, scroll）
- 使用 `requestAnimationFrame` 进行视觉更新

## 6. 响应式设计

### 6.1 断点定义

| 断点 | 宽度 | 适配策略 |
|------|------|----------|
| Mobile | < 768px | 简化动画，增大触摸目标 |
| Tablet | 768px - 1024px | 标准动画，侧边栏可折叠 |
| Desktop | > 1024px | 完整动画效果 |

### 6.2 移动端适配
- 触摸目标最小 44px
- 简化手势操作
- 禁用复杂悬停效果
- 启用触摸反馈

## 7. 无障碍设计

### 7.1 减少动画
```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

### 7.2 键盘导航
- 所有交互元素支持 Tab 导航
- 焦点状态清晰可见
- 支持 Enter/Space 激活

## 8. 实现技术方案

### 8.1 核心模块

```
src/modules/
├── animation/
│   └── AnimationEngine.js    # 动画引擎
├── effects/
│   └── VisualEffects.js      # 视觉效果
└── visualization/
    └── CanvasRenderer.js     # 渲染器（已集成动画）
```

### 8.2 使用示例

```javascript
// 创建弹簧动画
animationEngine.spring({
  target: node,
  property: 'scale',
  from: 0,
  to: 1,
  tension: 0.4,
  friction: 0.7,
  onUpdate: (value) => console.log(value)
});

// 创建涟漪效果
visualEffects.createRipple(x, y, {
  color: '#6366f1',
  maxRadius: 100,
  duration: 600
});

// 时间轴动画
const timeline = animationEngine.timeline();
timeline
  .add({ target: node1, property: 'x', from: 0, to: 100, duration: 300 })
  .add({ target: node2, property: 'y', from: 0, to: 100, duration: 300 })
  .play();
```
